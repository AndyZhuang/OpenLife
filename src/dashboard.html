📖 Getting Started

OpenLife is a bioinformatics AI agent built on ZeroClaw. Use the CLI or this dashboard to interact with bioinformatics skills.


🛠️ Installation

curl -L https://github.com/openlife-ai/openlife/releases/latest/download/openlife -o openlife
chmod +x openlife
sudo mv openlife /usr/local/bin/

💻 CLI Reference

openlife bio list              # List skills
openlife bio run        # Run a skill
openlife bio query "..."       # Natural language
openlife agent                 # AI Agent
openlife gateway               # Web dashboard
openlife version               # Version info
